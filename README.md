# mov2gif - convert mov files to gif

    $ git clone https://github.com/tumregels/mov2gif.git
    $ cd mov2gif
    $ npm install
    // put your mov file inside mov2gif folder and rename it into in.mov
    $ node index.js
    // previous command will generate out.gif
